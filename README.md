----

> 把垃圾文件放入回收站

适用于 Linux、macOS

**目前实现功能**

- [x] 直接删除文件
- [x] 直接回收文件
- [x] 基于用户自定义大小判断文件多大直接删除
- [ ] 撤销删除
- [ ] 清空回收站

### 安装

----

项目使用 Go 语言，可以直接下载发布的二进制文件，也可以使用下面的一件安装脚本

下载二进制文件放入 `/usr/loacl/bin` 目录下，把 `conf.yml` 文件放入 `$HOME/.config/goremove/` 这个路径下

### 使用方法

----

* 基于配置文件中定义的文件大小，如果需要删除的文件大于定义的大小，则会直接删除，无法恢复

```shell
grm [file]
```

* 直接删除文件，不论大小

```shell
grm -r [file]
```

* 直接把文件放入回收站，不论大小

```shell
grm -m [file]
```
